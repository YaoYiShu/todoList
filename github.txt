1.初始化本地仓库
git init

2.查看当前文件目录状态
git status

3.添加文件到暂存区
git add 文件名
git add * 添加全部

4.添加版本号
git commit -m '版本号(v1.0)'

5.查看日志记录
git log

6.回退版本（^代表上一个版本 ^^上上个版本）
git reset --hard HEAD^

7.回到某个版本 8ce5ae787844b3a48f6a0994f3a372423f3b7b01
git resset --hard id(前几位就行8ce5ae)

8.查看操作记录
git reflog

9.在未git add * 之前，查看新添加的内容部分
git diff

10.在未git add * 之前，执行撤销操作
git restore 文件

11.在git add * 添加到暂存区后，执行撤销操作
git restore --staged 文件

12.删除文件
git rm 文件

13.远程上传
git push

14.克隆项目
git clone http://------

15.继续推送更新的项目
git pull
